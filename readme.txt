== Description ==
This plugin allow to backup files and database and sends them to your email.
You can setup a periodically backup.

== Installation ==

1. Upload `backup2email.zip` to the `/wp-content/plugins/` directory
2. Unpack `backup2email.zip` to the current folder
3. Activate the plugin through the 'Plugins' menu in WordPress
